# vue-task-manager

A simple locally run task manager application completed to overview the Vue.js syntax.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve 
then
npm run backend
```

### Compiles and minifies for production
```
npm run build
```
